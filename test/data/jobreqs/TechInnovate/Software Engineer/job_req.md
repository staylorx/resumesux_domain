## Job Description

TechInnovate Inc. is seeking a passionate Senior Software Engineer to join our dynamic team. You'll be working on cutting-edge web applications using modern technologies like Flutter, Dart, and cloud services.

- Design and develop scalable web and mobile applications
- Collaborate with cross-functional teams including product managers and designers
- Implement clean, maintainable code following best practices
- Participate in code reviews and mentor junior developers

## Requirements

- 5+ years of experience in software development
- Proficiency in Dart and Flutter framework
- Strong knowledge of state management (Riverpod preferred)
- Experience with RESTful APIs and backend integration
- Familiarity with CI/CD pipelines and testing frameworks
- Bachelor's degree in Computer Science or related field

## Nice to Have

- Knowledge of clean architecture principles
- Contributions to open-source projects
- Agile development experience
